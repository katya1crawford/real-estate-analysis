Seismic/PML (Probable Maximum Loss) Report

If available, also obtain this report from the seller. It will indicate if the property is in an active seismic area and what, if any, damage was incurred in the last earthquake, as well as, what the probable maximum loss would be in the event of another strong earthquake. A PML of between 18 to 20 is acceptable to most lenders, over that creates cause for concern. This would be applicable to property located in California particularly.
