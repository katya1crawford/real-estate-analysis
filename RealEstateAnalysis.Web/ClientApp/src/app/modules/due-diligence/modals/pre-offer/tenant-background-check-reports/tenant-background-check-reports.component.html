All Current Tenants’ Applications & Background Check Reports

Purpose of this is to check if seller filled apartments with unqualified tenants in order to lower vacancy rate and increase value of the property before the sale. For all current tenants request from the seller tenant application and following background check reports:

  • Tenant Screening Report
  • Tenant Credit Report
  • Tenant Proof of Income Report
  • Criminal Background Check. Have they been completed on all residents?
  • Any other tenant background check reports
